Chezmoi Cheat Sheet

Essential commands for daily dotfile management

Key Concept

Source vs Target: Your source is in ~/.local/share/chezmoi/, your target is your actual dotfiles. chezmoi apply syncs source â†’ target.

Daily Editing

# Edit a config file (opens in nvim, auto-applies on save) chezmoi edit ~/.config/nvim/init.lua # Quick edit without auto-apply nvim ~/.config/nvim/init.lua chezmoi add ~/.config/nvim/init.lua # stage changes chezmoi apply # apply changes

Status & Sync

# See what's changed chezmoi diff chezmoi status # Apply all pending changes chezmoi apply # Update from source repo and apply chezmoi update

Adding New Files

# Add a new dotfile to chezmoi chezmoi add ~/.vimrc # Add entire directory chezmoi add ~/.config/alacritty

Templates

# View rendered template chezmoi cat ~/.config/jj/config.toml # Edit template source chezmoi edit ~/.config/jj/config.toml # Check template variables chezmoi data

Source Management

# Go to chezmoi source directory cd $(chezmoi source-path) # Commit changes (using jj in your setup) jj commit -m "Update nvim config" # View source vs target chezmoi diff

Useful Flags

  • --force Force apply (overwrite local changes)
  • --dry-run See what would happen without doing it
  • --verbose Show detailed output
chezmoi apply --force chezmoi apply --dry-run chezmoi apply --verbose

Emergency Recovery

# Restore from chezmoi if local file is broken chezmoi apply --force ~/.config/nvim/init.lua # See what chezmoi thinks a file should contain chezmoi cat ~/.bashrc
Emergency: If you break a config file, use chezmoi apply --force to restore it from your chezmoi source.
